Я люблю добротное кино

Рецензия на "Суперфорсаж!":

Очередной калоподобный фильмец от создателей «Вампирского засоса» и «Знакомства со спартанцами» - «Суперфорсаж», вобравший в себя все «лучшее» из знаменитой франшизы. История пародийного фильма до боли напомнит зрителю о славных временах первого «Форсажа», когда Уокер познакомился с Торетто и его семьей, о том, как во франшизе появился Скала Джонсон и что творилось в последующих фильмах (тут я имею ввиду сцену с ограблением и гигантским сейфом). Вот честно, не знаю, что побудило посмотреть этот фильм, но просто взял и включил. И Посмотрел. Господи! Я понимаю, что это фильм-пародия, но как?! Как можно снимать такую ересь, такое дерьмо. Не побоюсь этого слова. И как название «Суперфорсаж» пересекается с сюжетом? Где тут форсаж и тем более супер? За 100 минут экранного времени можно наблюдать за одним гоночным заездом и плоскодонными погонями в формате преступники-полиция. Пройдясь катком по фильмам и взяв из них все что только можно – китайцев, тэпэшек, негров, типа крутые тачки, мускулы Торетто и прическу Уокера, создатели «Суперфорсажа» напичкали кино тупым шаблонным юмором, в котором абсолютно нет ничего смешного. Ровным счетом ничего. Даже дубляж, сделанный в пользу российского зрителя с упоминанием нашего «достояния» в виде Тимати, а еще шоу «Голос» и Градского – вариант оправданный, но никак не позволяющий хотя бы на «+1» повысить юморную составлющую «Суперфорсажа». В принципе, довольно крупный бюджет в 20 миллионов….Как? На что, извините, были потрачены эти бабки? На приобретение двух силиконовых имплантов? Или на прокачку мышц Вина Дросселя? Или на наклейки для спорткаров? Как можно было вбухать в Это столько бабла? И зачем, спрашивается? Понятно, что режиссеры приложили руку к Scary Movie, но там хоть и сюжет был смотрибелен (по крайней мере в 1-2 частях), и пародия охватывала куда большее количество фильмов. А здесь: тупые персонажи, отвратные гонки, тупые персонажи, и нарезка неудачных кадров – единственное, на что, на мой взгляд, здесь можно обратить внимание. 1 из 10

Я умею писать код

Пример кода:

                
                    import React, {useEffect} from "react";
                    import {connect} from "react-redux";
                    import {Route} from "react-router-dom";

                    import TasksList from "../tasks-list/tasks-list";
                    import AppHeader from "../app-header/app-header";
                    import Contacts from "../contacts/contacts";
                    import NewTaskForm from "../new-task-form/new-task-form";
                    import Filter from "../filter/filter";

                    import './app.scss';
                    import {Container} from "reactstrap";

                    const App = (props) => {
                    useEffect(() => {
                    if (localStorage['lang'] === undefined) {
                    localStorage['lang'] = 'English';
                    }
                    if (localStorage['theme'] === undefined) {
                    localStorage['theme'] = 'Light';
                    }
                    }, [])

                    return (
                    <div className={props.theme === 'Light' ? "app" : "app app_dark"}>
                    <AppHeader/>
                    <Route exact path="/todo-list-react">
                        <Container>
                            <NewTaskForm/>
                            <Filter/>
                            <TasksList/>
                        </Container>
                    </Route>
                    <Route path="/todo-list-react/contacts" component={Contacts} title="Contacts"/>
                    </div>
                    );
                    }

                    const mapStateToProps = (state) => {
                    return {
                    theme: state.theme
                    }
                    }

                    export default connect(mapStateToProps)(App);
